## 🚀 快速开始

### 1. 环境要求

- PHP 7.4 或更高版本
- MySQL 5.7 或更高版本
- 支持 PDO 扩展
- 可写的文件上传目录

### 2. 安装步骤

1. **上传文件**
   将所有文件上传到你的 web 服务器根目录。

2. **访问网站**
   在浏览器中访问你的网站 URL：
   ```
   https://你的域名
   ```

3. **运行安装向导**
   首次访问会自动跳转到安装页面，完成以下步骤：
   
   - **步骤 1**: 填写数据库连接信息
     - 数据库主机
     - 数据库名
     - 数据库用户名
     - 数据库密码
     
   - **步骤 2**: 创建管理员账号
     - 管理员用户名
     - 管理员密码

4. **完成安装**
   点击"开始安装"，系统会自动：
   - 测试数据库连接
   - 创建数据表结构
   - 生成配置文件
   - 创建管理员账号
   - 跳转到登录页面


### 3. 登录系统

1. **访问登录页面**
   ```
   https://你的域名/login.php
   ```

2. **输入管理员账号密码**
   使用安装时设置的管理员账号和密码登录。

3. **开始使用**
   登录后即可开始管理文件和文件夹。
